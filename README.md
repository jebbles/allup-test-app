# [allUP Video Playlist Demo App](https://duckduckgo.com](https://jebbles.github.io/allup-test-app/)

## Setup

1. Clone this repo
```
git clone https://github.com/jebbles/allup-test-app.git
cd allup-test-app
```
2. Install dependencies 
```
npm install
```
4. Run app locally 
```
npm run dev
```
6. View site on browser
```
localhost:xxxx
```
